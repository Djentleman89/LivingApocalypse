# LivingApocalypse

Vanilla 7D2D zombie counts don’t seem very…apocalyptic. So I created this mod to change that and make the world feel far more alive, dangerous, and unpredictable over future releases, starting with the current feature:

Migration Events. While traveling, you may come across groups of zombies migrating across the roads and in the wilderness. No, this isn’t another “spawn 20 and attack the player” type mod. They’re designed to feel as organic as possible; imagine you crest a hill and see a group crossing the road ahead of you. You can either wait, fight or detour. But don’t get too close…

Because my computer isn’t the best, I made sure to optimize it as well as I can. Spawning is staggered to reduce stutter and despawn rules are set to ensure there’s not a massive entity load. 

FEATURES:\
-Natural spawning based on player movement and direction\
-Early game protection with smaller migration groups, so you don’t accidentally have to fight with a level 1 bow and stone arrows lol\
-Larger groups unlocked later in progression, along with biome specific migrations.\
-Built with the thought of “How can I make this the most organic so it doesn’t feel cheap”\
